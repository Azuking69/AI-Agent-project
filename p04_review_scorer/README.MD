# 🍰 グルメ本音ラボ / 그루메 혼네 라보

## 📌 コンセプト / 컨셉

現地の人しか知らない「本音」を、日本語でそのまま届けたい。

韓国のレビューサイト（네이버 지도・카카오맵など）には、現地の人の生の声がたくさん眠っています。でも機械翻訳にかけると、料理名が変な訳になったり、ニュアンスが伝わらなかったり…。

「グルメ本音ラボ」は、そんな**日本人旅行者**のために作った、韓国語レビューの構造化・翻訳ツールです。ただ翻訳するだけでなく、星評価・感情・要約・キーワードに整理することで、旅行者が一瞬で「行くべきか」を判断できるようにします。

한국의 리뷰 사이트（네이버 지도・카카오맵 등）에는 현지인의 생생한 목소리가 많이 담겨 있습니다. 하지만 기계 번역을 거치면 요리명이 이상하게 번역되거나 뉘앙스가 전달되지 않는 경우가 많습니다.

"그루메 혼네 라보"는 그런 **일본인 여행자**를 위해 만든, 한국어 리뷰 구조화·번역 도구입니다. 단순 번역이 아니라 별점·감정·요약·키워드로 정리하여, 여행자가 한눈에 "가볼 만한 곳인지" 판단할 수 있도록 돕습니다.

---

## 📌 概要 / 개요

Flask + Anthropic SDK（Structured Outputs）を使った、飲食店レビューを星・感情・要約・キーワードに構造化して評価するツールです。
Flask + Anthropic SDK（Structured Outputs）를 사용한, 음식점 후기를 별점·감정·요약·키워드로 구조화해 평가하는 도구입니다.

---

## ✨ 機能 / 기능

- 📄 レビュー`.txt`ファイルのアップロード、または直接貼り付け / 후기 `.txt` 파일 업로드 또는 직접 붙여넣기
- ⭐ LLMによる星評価（1〜5）・感情判定（positive/neutral/negative） / LLM에 의한 별점(1~5)·감정 판정
- 📝 一文要約・根拠キーワードの抽出 / 한 문장 요약·근거 키워드 추출
- 🛡️ Pydanticによる出力スキーマ検証（Structured Outputs） / Pydantic을 통한 출력 스키마 검증
- 🌏 韓国語レビューを日本語に翻訳しつつ、料理名などの固有名詞はカタカナ音写 / 한국어 후기를 일본어로 번역하면서, 요리명 등 고유명사는 가타카나로 발음 표기
- 🧳 辛さ・混雑状況・外国人客への対応など、旅行者が気にする実用情報を優先的に拾う / 맵기·혼잡도·외국인 응대 등 여행자가 궁금해할 실용 정보를 우선적으로 반영
- ⚠️ 失敗（空・長すぎ・検証失敗・通信エラーなど）を例外ではなく値として処理 / 실패（빈 값·과도한 길이·검증 실패·통신 오류 등）를 예외가 아닌 값으로 처리

## 🌐 実装したルート / 구현한 라우트

| ルート    | メソッド | 内容                                                                                  |
| --------- | -------- | ------------------------------------------------------------------------------------- |
| /         | GET      | レビュー入力画面を表示 / 후기 입력 화면 표시                                          |
| /evaluate | POST     | レビューを受け取りLLM評価結果をJSONで返す / 후기를 받아 LLM 평가 결과를 JSON으로 반환 |

### POST /evaluate リクエスト例

```json
{
  "review": "사장님이 서비스로 계란찜을 내주셨어요. 정도 있고 음식도 하나하나 정갈합니다. 오늘부로 동네 단골집 등극이네요!"
}
```

### レスポンス（成功）

```json
{
  "ok": true,
  "evaluation": {
    "score": 5,
    "sentiment": "positive",
    "summary": "親切な店主のサービスと、丁寧で質の高い料理に満足し、常連店として通う決意をしている。",
    "keywords": ["サービスで卵蒸し", "料理が丁寧", "常連店に認定"]
  }
}
```

### レスポンス（失敗）

```json
{ "ok": false, "error": "empty" }
```

### エラーコード一覧 / 에러 코드 목록

| コード / 코드  | 内容 / 내용                                               |
| -------------- | --------------------------------------------------------- |
| empty          | レビューが空 / 후기가 비어있음                            |
| too_long       | 入力が長すぎる / 입력이 너무 길다                         |
| invalid_output | LLM出力がスキーマ検証に失敗 / LLM 출력이 스키마 검증 실패 |
| truncated      | max_tokensで応答が途中で切れた / max_tokens로 응답이 잘림 |
| connection     | ネットワーク・タイムアウトエラー / 네트워크·타임아웃 오류 |
| server_error   | APIの5xxエラー / API 5xx 오류                             |
| bad_request    | リクエスト形式エラー / 요청 형식 오류                     |

## 🩹 トラブルシューティング / 트러블슈팅

開発中に遭遇した問題とその対応をまとめました。

| 症状 / 증상                                                                     | 原因 / 원인                                                 | 対応 / 대응                                                                    |
| ------------------------------------------------------------------------------- | ----------------------------------------------------------- | ------------------------------------------------------------------------------ |
| キーワードが韓国語のまま出力される                                              | System Promptに出力言語の指定がなかった                     | 「出力は日本語で」という指示を追加                                             |
| 固有名詞（料理名）が誤訳される（例：불고기→不動）                               | 「翻訳しろ」という指示だけでは意味翻訳されてしまう          | 「固有名詞はカタカナ音写する」という指示を追加（例：불고기→プルコギ）          |
| 単語の一部だけ翻訳され、他が原語のまま残る（例：계란찜→「サービスで계란蒸し」） | 一般名詞的な料理名で、音写と意訳のどちらにするかLLMが迷った | 「単語全体を訳すか、単語全体を音写するか、どちらかに統一する」という指示を追加 |

## 📝 元のコードからの変更点 / 원본에서 바꾼 점

- UIをすべて日本語化（元は韓国語UI） / UI를 전부 일본어로 변경（원본은 한국어 UI）
- デザインをオリジナルの「ショートケーキ×実験ラボ」テイストに変更 / 디자인을 오리지널 "쇼트케이크×실험 랩" 테이스트로 변경
- Evaluationモデルのフィールド名を韓国語（점수・감정・요약・키워드）から英語（score・sentiment・summary・keywords）に変更 / Evaluation 모델의 필드명을 한국어에서 영어로 변경
- SYSTEM_PROMPTを日本語に翻訳し、日本人旅行者向けの評価基準（旅行者目線の要約・実用情報の優先・固有名詞の扱い）にアレンジ / SYSTEM_PROMPT를 일본어로 번역하고, 일본인 여행자를 위한 평가 기준（여행자 시점 요약·실용 정보 우선·고유명사 처리）으로 조정
- コンセプトを「レビュー評価器」から「日本人旅行者向けの韓国語レビュー翻訳・診断ツール」に再定義 / 컨셉을 "후기 평가기"에서 "일본인 여행자를 위한 한국어 후기 번역·진단 도구"로 재정의
